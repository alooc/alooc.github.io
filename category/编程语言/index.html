    疯狂前端开发讲义笔记

    Ajax 即“Asynchronous Javascript And XML”（异步 JavaScript 和 XML），
    是指一种创建交互式、快速动态网页应用的网页开发技术，
    无需重新加载整个网页的情况下，
    能够更新部分网页的技术。

    XMLHttpRequest是Ajax的核心，JavaScript是Ajax技术的黏合剂

    整个Ajax应用的工作过程：
    1.JavaScript脚本使用XMLHttpRequest对象向服务器发送请求。
    发送请求时，既可以发送GET请求，也可以发送POST请求。
    2.JavaScript脚本使用XMLHttpRequest对象解析服务器响应数据。
    3.JavaScript脚本通过DOM动态更新HTML页面。也可以为服务器响应数据增加CSS样式表，
    在当前网页的某个部分加以显示。
