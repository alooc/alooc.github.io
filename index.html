    # 🚀

    # 美团 Java开发高级工程师 25-50K·15薪

    1. 计算机相关专业本科以上学历，3年以上Java开发工作经验，具备较强的开发能力，注重代码质量，有良好的软件工程知识和编码规范意识；
    2. 具有扎实的Java功底，了解JVM的原理，具有较好JavaIO、多线程、网络等方面的编程能力；
    3. 了解 SOA 架构理念、实现技术；熟悉常见设计模式，精通SpringMvc、MyBatis等主流开源框架，熟悉常用第三方类库；
    4. 精通数据库设计（MySQL/Oracle），优秀的SQL编写及调优能力，熟悉常见NoSQL存储，如Hbase、Memcached、Redis、Mongodb等；
    5. 熟悉Linux平台下常用命令操作、环境部署；
    6. 有强烈的责任心和团队合作精神，良好的抗压能力，心态积极，能主动融入团队；
    7. 具有较好的沟通能力，思路清晰，善于思考，能独立分析和解决问题；
    8. 能够独当一面，独立负责模块/子系统的业务理解、系统设计、开发测试，运维调优；
    9. B端业务系统，要求有复杂业务的开发经验和较强的逻辑/概率思维能力，善于分析、归纳、描述、沟通、和解决问题；

    # 快手 高级Java研发工程师

    1. 3年以上工作经验，计算机或相关专业毕业，有扎实的计算机基础知识和编程功底，良好的编码习惯；
    2. 精通Java，熟悉MySQL、Memcache、Redis等常用开源软件，掌握HTTP、TCP等协议，对性能调优、故障处理等有较强的分析和处理能力；
    3. 积极主动，有Owner意识，沟通顺畅，有良好的抗压能力，对技术有热情，喜欢钻研，学习能力强，乐于分享；
    4. 有较强的团队协作能力和合作意识，能站在全局角度思考最佳解决方案，并推动落地；
    5. 有过运营类系统研发或海量数据处理经验优先。

    # 阿里 Java高级工程师/技术专家 20-40K

    1. 本科学历, 互联网/软件行业3年以上工作经验；
    2. 优秀的分析问题和解决问题的能力, 敢于面对业务和技术双重挑战, 对解决具有挑战性问题充满激情, 抗压能力好；
    3. 熟悉各种架构模式（缓存，分布式，Ioc，DDD，CQRS等）设计模式，编码模式，以及相应的反模式；
    4. 熟悉各种编程范式
    5. 熟悉应用架构设计，善于划分API职责与子系统；
    6. 熟悉自动化测试和测试驱动设计；
    7. 熟悉网络编程、多线程编程技术，有大规模系统开发和设计经验；
    8. 精通linux平台上的Java语言编程，良好的编码习惯，熟悉数据库开发技术，熟悉shell编程；
    9. 对数据结构、算法设计、系统架构设计等有较为深刻的理解, 有异步架构开发经验优先；
    10. 有系统监控、数据分析模块、开放平台、实时计算等核心系统工作经验，有Hadoop等海量数据研发经验优先；
    11. 良好的沟通能力，善于学习, 乐于分享，有多语言技术栈, 参与过开源项目开发者优先。
