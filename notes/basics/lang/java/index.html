访问修饰符 ​

  • private 仅本类访问
  • default 本类及类所在包可访问
  • protected 本类，类所在包，继承当前类的子类可访问
  • public 所有类都可访问 接口的方法默认是public

==和equals ​

==： 判断两个对象的地址是不是相等，基本数据类型比较值是否相等

equals：判断两个对象的内容是否相等

​ 若重写equals,需要重写hascode。规定，两个对象相等，则hascode也一定相同。若不重写，会出现两个hascode相同的对象，但不是equals
